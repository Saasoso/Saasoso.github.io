JavaScript

JavaScript est un langage de programmation couramment utilisé dans le développement web. Il est souvent utilisé pour ajouter un comportement interactif aux pages web et est pris en charge par tous les navigateurs web modernes.

Variables

Les variables sont utilisées pour stocker et manipuler des données en JavaScript. Vous pouvez déclarer une variable en utilisant le mot-clé `var`, comme ceci:

  
  var message;

Vous pouvez également affecter une valeur à une variable lorsque vous la déclarez:

  
  var message = "Bonjour le monde!";

Vous pouvez également utiliser les mots-clés `let` et `const` pour déclarer des variables. `let` est similaire à `var`, mais a des règles de portée plus strictes. `const` est utilisé pour déclarer des variables qui ne peuvent pas être réaffectées, comme ceci:

  
  const pi = 3.14;

Types de données

JavaScript a plusieurs types de données, notamment:

  • chaînes de caractères (par ex. `"bonjour"`)
  • nombres (par ex. `42`)
  • booléens (par ex. `true` ou `false`)
  • tableaux (par ex. `[1, 2, 3]`)
  • objets (par ex. `{nom: "Saad", age: 30}`)

Vous pouvez utiliser l'opérateur `typeof` pour obtenir le type de données d'une valeur, comme ceci:

  
  document.write(typeof "bonjour"); --> "string"
  document.write(typeof 42); -->"number"
  document.write(typeof true); --> "boolean"
  document.write(typeof [1, 2, 3]); --> "object"
  document.write(typeof {nom: "Saad", age: 30}); --> "object"

Opérateurs

JavaScript possède de nombreux opérateurs qui peuvent être utilisés pour effectuer des opérations sur des valeurs. Certains opérateurs courants sont:

  • opérateurs arithmétiques (par ex. `+`, `-`, `*`, `/`)
  • opérateurs de comparaison (par ex. `==`, `!=`, `>`, `<`)
  • opérateurs logiques (par ex. `&&`, `||`, `!`)

Par exemple, vous pouvez utiliser l'opérateur `+` pour additionner deux nombres:

  
  document.write(1 + 1); --> Affiche : 2

Vous pouvez également utiliser des opérateurs de comparaison pour comparer des valeurs. Par exemple:

  
  document.write(1 == 1); --> Affiche : true
  document.write(1 == 2); --> Affiche : false
  document.write(1 != 2); --> Affiche : true
  document.write(1 > 2); --> Affiche : false
  document.write(1 < 2); --> Affiche : true

Structures de contrôle

JavaScript possède plusieurs structures de contrôle qui vous permettent de contrôler le déroulement de votre code. Certaines structures de contrôle courantes sont:

  • instructions `if`
  • boucles `for`
  • boucles `while`

Par exemple, vous pouvez utiliser une instruction `if` pour exécuter un bloc de code uniquement si une certaine condition est remplie:

  
  var x = 10;
  if (x > 5) {
    document.write("x est supérieur à 5");
  }

Vous pouvez également utiliser une boucle `for` pour parcourir un tableau:

  
  var numbers = [1, 2, 3, 4, 5];
  for (var i = 0; i < numbers.length; i++) {
    document.write(numbers[i]);
  }

Événements

JavaScript vous permet de rattacher du code à des événements déclenchés par l'utilisateur ou le navigateur. Par exemple, vous pouvez utiliser l'événement `onclick` pour exécuter une fonction lorsqu'un bouton est cliqué:

    
    <button onclick="saluer('Saad')">Cliquez ici!</button>
    
    
        
        document.getElementById("mon-bouton").addEventListener("click", function() {

        });

Il existe de nombreux autres types d'événements que vous pouvez utiliser, tels que:

  • `load`: lorsque la page est chargée
  • `mouseover`: lorsque la souris entre dans la zone d'un élément
  • `submit`: lorsqu'un formulaire est soumis
  • `keydown`: lorsqu'une touche du clavier est enfoncée

Notez que la fonction doit être définie avant d'être appelée dans l'attribut `on`. Vous pouvez également utiliser l'objet `event` dans la fonction pour accéder à des informations sur l'événement qui s'est produit

Objets

En JavaScript, un objet est une collection de propriétés qui peuvent être de différents types (nombres, chaînes de caractères, fonctions, etc.). Vous pouvez définir un objet en utilisant la syntaxe suivante:

  
  var objet = {
    propriété1: valeur1,
    propriété2: valeur2,
    ...
  };

Par exemple, vous pouvez définir un objet "personne" qui a des propriétés "nom" et "âge":

  
  var personne = {
    nom: "Saad",
    âge: 30
  };

Vous pouvez également définir une fonction dans un objet, appelée "méthode":

  
  var personne = {
    nom: "Saad",
    âge: 30,
    sePresenter: function() {
      document.write("Bonjour, je m'appelle " + this.nom + " et j'ai " + this.âge + " ans.");
    }
  };
  personne.sePresenter(); --> Affiche : Affiche "Bonjour, je m'appelle Jean et j'ai 30 ans."

Tableaux

Un tableau en JavaScript est une collection d'éléments qui peuvent être de différents types. Vous pouvez définir un tableau en utilisant la syntaxe suivante:

  
  var tableau = [élément1, élément2, ...];

Par exemple, vous pouvez définir un tableau de noms:

  
  var noms = ["S", "C", "c"];

Vous pouvez accéder à un élément d'un tableau en utilisant son indice (l'emplacement de l'élément dans le tableau), commençant à 0:

  
  document.write(noms[0]); --> Affiche S
  document.write(noms[1]); --> Affiche C
  document.write(noms[2]); --> Affiche c

Vous pouvez également utiliser la méthode `push` pour ajouter un élément à la fin d'un tableau:

  
  noms.push("Paul");
  document.write(noms); --> Affiche ["S", "a", "a", "d"]
class Personne { constructor(nom, âge) { this.nom = nom; this.âge = âge; } sePresenter() { document.write("Bonjour, je m'appelle " + this.nom + " et j'ai " + this.âge + " ans."); } }

Vous pouvez créer une instance de la classe "Personne" en utilisant la syntaxe `new`:

  
  var personne1 = new Personne("Saad", 30);
  personne1.sePresenter(); --> Affiche : Affiche "Bonjour, je m'appelle Jean et j'ai 30 ans."

Vous pouvez également définir une classe en étendant (héritant de) une autre classe:

  
  class Etudiant extends Personne {
    constructor(nom, âge, matricule) {
      super(nom, âge); --> Affiche : Appel du constructeur de la classe parente
      this.matricule = matricule;
    }
    sePresenter() {
      document.write("Bonjour, je m'appelle " + this.nom + " et j'ai " + this.âge + " ans. Mon matricule est " + this.matricule + ".");
    }
  }

Vous pouvez créer une instance de la classe "Etudiant" de la même manière que pour la classe "Personne":

  
  var etudiant1 = new Etudiant("Saad", 20, "abc123");
  etudiant1.sePresenter(); --> Affiche : Affiche "Bonjour, je m'appelle Saad et j'ai 20 ans. Mon matricule est abc123."
